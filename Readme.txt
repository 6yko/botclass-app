BotClass.Desktop - программа для взаимодействия с API веб-приложения botclass.ru
Source_code содержит исходный код написанный при помощи .NET 8.0 и Avalonia UI в качестве кросплатформенного фреймворка.
IDE для разработки - Microsoft Visual Studio 2022 
Release содержит билды для MacOS (osx-x64) и Windows (win-x64)

Инструкция для MacOS:
Распаковать папку BotClass.Desktop.app на рабочий стол (или другое место, но изменить путь в следующей команде)
Открыть терминал и ввести следующую команду для разрешения на исполнение:
chmod +x ~/Desktop/BotClass.Desktop.app/Contents/MacOS/BotClass.Desktop

Для операционных систем версий <10.15 открыть окно безопасности и разрешить запуск приложений из любого источника.
Для операционных систем Catalina и выше необходимо отключить GateKeeper.

Команда терминала:
sudo spctl --master-disable
Или
sudo spctl --global-disable

После чего запустить приложение, выдать разрешение для запуска хука клавиатуры (отслеживающий горячую комбинацию) и перезапустить приложение.
По-умолчанию приложение сворачивается при попытке закрыть его, для выхода воспользуйтесь ПКМ по иконке в трее, либо отключить Always Minimaze to Tray в настройках.
После проделанных манипуляция можно будет пользоваться приложением.
Если после отключения GateKeeper'а операционная система все равно не разрешает запустить приложение нажмите ПКМ -> Открыть -> Открыть

Приложение не подписано сертификатом разработчика Apple и не нотаризировано, поэтому требуется отключать хранителя!

Для Windows:
Выберите понравившийся билд и запустите исполняемый файл .exe
